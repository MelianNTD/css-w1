# Dự Án Quản Lý Giải Đấu Bóng Đá (css-w1)

Đây là dự án bài tập tuần 1 tập trung vào việc xây dựng giao diện website bóng đá bằng HTML và CSS.

- **Trang chủ/Trận đấu:** Hiển thị danh sách và kết quả các trận đấu (`matches.html`).
- **Đội bóng:** Thông tin về các đội bóng tham gia giải đấu (`teams.html`).
- **Top cầu thủ:** Bảng xếp hạng các cầu thủ xuất sắc nhất (`Topcauthu.html`).

## 🛠 Công nghệ sử dụng
- **HTML5:** Cấu trúc nội dung trang web.
- **CSS3:** Định dáng giao diện, layout và màu sắc (lưu trữ trong thư mục `/css`).

## 📁 Cấu trúc thư mục
- `/css`: Chứa các file định dạng giao diện (`style.css`).
- `/images`: Chứa hình ảnh sử dụng trong giao diện.
- Các file `.html` nằm ở thư mục gốc.
